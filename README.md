Clone the project

```bash
  git clone https://github.com/JKaliraj/ZeroNotes.git
```

Go to the project directory

```bash
  cd ZeroNotes/
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python manage.py runserver
```

Open in browser

```bash
  http://127.0.0.1:8000/
```
